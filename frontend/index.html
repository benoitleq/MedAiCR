💾 Sauvegarde / restauration de la configuration

Exporte (ou réimporte) toute ta configuration dans un seul fichier : dossiers surveillés, réglages IA + prompts système + clé API, et types d'examens personnalisés. Pratique après une réinstallation.

⚠ Le fichier de sauvegarde contient ta clé API en clair — conserve-le en lieu sûr.

⚙ Surveillance automatique de dossiers

Définis les dossiers à surveiller. Dès qu'un PDF y est déposé, une version ANOM_…pdf est créée à côté (même mise en page). Les modifications sont appliquées automatiquement par le service de surveillance.

🧠 Génération de CR (IA)

Génère un compte rendu à partir du texte anonymisé. Choisis ton fournisseur d'IA, sa clé API et son modèle, puis colle ton « system prompt » (l'équivalent d'un GEM Gemini / GPT) pour chaque type d'examen.

C'est ce fournisseur (une fois enregistré) qui sera appelé quand tu cliques « Générer le CR ».
⚠ La génération envoie le texte anonymisé au fournisseur d'IA choisi (service externe). Vérifie le récapitulatif d'anonymisation avant d'envoyer. Les clés API sont stockées localement sur ce poste. La génération utilise le fournisseur enregistré ici.

➕ Ajouter un type de document (assisté par IA)

Fournis un CR FICTIF (faux patient) du nouveau type. L'IA repère les champs identifiants et propose les règles ; elles tournent ensuite 100 % en local. ⚠ N'utilise jamais un vrai patient ici.

Astuce : nom ZZNOMTEST, prénom ZZPRENOMTEST, n° 0000000001, naissance 01/01/1900 (valeurs distinctives → vérification fiable).
